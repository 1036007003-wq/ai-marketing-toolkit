# 🚀 AI Marketing Automation Bundle - User Guide

6 production-ready MCP (Model Context Protocol) servers that let AI agents automate your marketing across every major platform:

1. **Reddit Marketing MCP** - Research subreddits, find trending posts, analyze competitors, draft posts
2. **Twitter Marketing MCP** - Search tweets, analyze engagement, draft replies, track metrics

## Quick Start (5 minutes)

### Step 1: Install Prerequisites

```bash
# Install Node.js (if not installed)
# macOS: brew install node
# Windows: Download from https://nodejs.org

# Install Claude Desktop or Cline (VS Code extension)
# Claude Desktop: https://claude.ai/download
# Cline: Install from VS Code marketplace
```

### Step 2: Set Up Each MCP Server

```bash
# Clone all repositories
git clone https://github.com/1036007003-wq/reddit-marketing-mcp.git
git clone https://github.com/1036007003-wq/twitter-marketing-mcp.git
git clone https://github.com/1036007003-wq/discord-ai-mcp.git
git clone https://github.com/1036007003-wq/github-stars-mcp.git
git clone https://github.com/1036007003-wq/linkedin-marketing-mcp.git
git clone https://github.com/1036007003-wq/tiktok-marketing-mcp.git

# Install dependencies for each
cd reddit-marketing-mcp && npm install && cd ..
cd twitter-marketing-mcp && npm install && cd ..
cd discord-ai-mcp && npm install && cd ..
cd github-stars-mcp && npm install && cd ..
cd linkedin-marketing-mcp && npm install && cd ..
cd tiktok-marketing-mcp && npm install && cd ..
```

### Step 3: Configure Claude Desktop

Edit `~/Library/Application Support/Claude/claude_desktop.json` (macOS) or `%APPDATA%\Claude\claude_desktop.json` (Windows):

```json
{
  "mcpServers": {
    "reddit-marketing": {
      "command": "node",
      "args": ["/full/path/to/reddit-marketing-mcp/index.js"]
    },
    "twitter-marketing": {
      "command": "node",
      "args": ["/full/path/to/twitter-marketing-mcp/index.js"]
    },
    "discord-ai": {
      "command": "node",
      "args": ["/full/path/to/discord-ai-mcp/index.js"]
    },
    "github-stars": {
      "command": "node",
      "args": ["/full/path/to/github-stars-mcp/index.js"]
    },
    "linkedin-marketing": {
      "command": "node",
      "args": ["/full/path/to/linkedin-marketing-mcp/index.js"]
    },
    "tiktok-marketing": {
      "command": "node",
      "args": ["/full/path/to/tiktok-marketing-mcp/index.js"]
    }
  }
}
```

### Step 4: Restart Claude Desktop

Close and reopen Claude Desktop. The MCP servers will now be available.
