💬 **Discord AI MCP** — find servers, draft community messages, analyze engagement
🐙 **GitHub Stars MCP** — analyze star growth, optimize README, find similar repos

**ALL FREE** (no API key needed for basic features). Premium features (AI-powered drafts) use DeepSeek API (~$0.14/1M tokens).

👉 **Main repo (all 4 tools):** https://github.com/1036007003-wq/ai-marketing-toolkit

Individual repos:
- Reddit: https://github.com/1036007003-wq/reddit-marketing-mcp
- Twitter: https://github.com/1036007003-wq/twitter-marketing-mcp
- Discord: https://github.com/1036007003-wq/discord-ai-mcp
- GitHub: https://github.com/1036007003-wq/github-stars-mcp

Feedback welcome! What other marketing tasks should I automate next?

## 第5步：开启GitHub Sponsors（15分钟，必须本人做）

### 打开：https://github.com/settings/billing

### 找到 **"GitHub Sponsors"** 那一行 → 点 **"Enable Sponsors"**

### 填写：

**Tier 1:** $5/month → Description: `Unlock all premium AI features`

**Tier 2:** $25/month → Description: `All premium + priority support + custom AI prompts`

**Tier 3:** $99 (One-time) → Description: `Lifetime access (no subscription)`

### 绑定收款方式：

**选项A（推荐）：PayPal**
1. 点 **"Connect PayPal"**
2. 登录PayPal，授权
3. 完成

**选项B：Bank account（银行卡）**
1. 点 **"Add bank account"**
2. 填银行卡信息
3. 点 **"Save"**

### 点 **"Save changes"**
